# AI 분석 결과 확인 방법

## 📍 분석 결과가 나타나는 곳

**브라우저 개발자 도구의 콘솔(Console) 탭**에 출력됩니다!

## 🎯 콘솔 확인 방법

### 1. 개발자 도구 열기

**Windows:**
- `F12` 키 누르기
- 또는 `Ctrl + Shift + I`

**Mac:**
- `Cmd + Option + I`

### 2. Console 탭 선택

개발자 도구가 열리면:
- 상단의 **"Console"** 탭 클릭
- 콘솔 탭이 보이지 않으면 `Esc` 키를 눌러 사이드 패널 열기

## 🔍 분석 결과 형식

에러가 발생하고 분석이 완료되면 콘솔에 다음과 같이 표시됩니다:

```
======================================================================
                    🤖 AI 에러 분석 결과                    
======================================================================

1) 에러 원인
RangeError: Invalid array length

2) 왜 발생했는가
배열의 길이로 음수(-1)를 지정하려고 했기 때문에 발생했습니다.
배열 길이는 0 이상의 정수여야 합니다.

3) 해결방법 (코드 예시 포함)
음수 대신 올바른 배열 길이를 사용하세요:
// 잘못된 예:
var arr = new Array(-1);

// 올바른 예:
var arr = new Array(10); // 10개 요소
// 또는
var arr = []; // 빈 배열

4) 고객 안내 멘트
이 오류는 개발 과정에서 발생한 것으로, 곧 수정될 예정입니다.

======================================================================
💡 팁: 브라우저 개발자 도구(F12) → Console 탭에서 이 메시지를 확인할 수 있습니다.
======================================================================
```

## ⚠️ 중요 사항

### 1. 콘솔을 열어두어야 합니다

에러가 발생한 후 콘솔을 열어도 이전 메시지를 볼 수 있지만, 
**에러 발생 전에 콘솔을 열어두는 것을 권장**합니다.

### 2. 분석 시간

- **5-10초 정도** 소요될 수 있습니다
- 분석이 시작되면 콘솔에 "🔍 에러 분석 시작..." 메시지가 표시됩니다
- 분석 완료 후 결과가 자동으로 출력됩니다

### 3. 콘솔 필터

콘솔에 많은 메시지가 있다면:
- 콘솔 상단의 필터에서 `[AI Error Assistant]` 검색
- 또는 "Errors" 필터 선택

## 📝 테스트 방법

1. **F12**로 개발자 도구 열기
2. **Console** 탭 선택
3. 에러 발생시키기 (버튼 클릭 또는 코드 실행)
4. 콘솔에서 분석 결과 확인

## 💡 빠른 확인

브라우저 콘솔에서 직접 확인:

```javascript
// 콘솔을 이미 열어둔 상태에서
testReferenceError(); // 에러 발생 및 자동 분석

// 또는
undefinedFunction(); // 에러 발생 및 자동 분석
```

## 🎨 콘솔 메시지 색상

- **파란색**: 분석 시작 알림
- **주황색**: 진행 중 알림
- **초록색**: 분석 결과 박스
- **일반 텍스트**: 분석 내용

## ❓ 결과가 안 보이면

1. **콘솔 탭이 선택되어 있는지** 확인
2. **콘솔 필터**가 모든 메시지를 표시하도록 설정
3. **스크롤을 올려서** 이전 메시지 확인
4. **콘솔 지우기** 후 다시 테스트
